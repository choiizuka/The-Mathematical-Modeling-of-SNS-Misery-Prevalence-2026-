* **$D(\tau)$ (Defamation Density):** 単位時間あたりの誹謗中傷・ネガティブノイズの密度。
* **$V(\tau)$ (Velocity of Spread):** アルゴリズムによる拡散速度。インプレッション増幅係数。
* **$\Phi(S)$ (Sensitivity of Sanctuary):** 被害側の感受性および過去のトラウマ的背景（$S$）に基づく増幅関数。
* **$\alpha$ (Social Amplification Factor):** 社会的情勢（不景気、政治的分断など）による、攻撃性の社会的な増幅定数。
